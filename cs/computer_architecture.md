cpu 내부에는 다양한 레지스터 들이 있고 각기다른 역할을 가진다.

1. 프로그램 카운터 
-메모리에서 가져올 명령어의 주소
-여기로 읽은 메모리 주소 값은 메모리 주소 레지스터로 복사됨
2. 명령어 레지스터
-해석할 명령어 (방금 메모리에서 읽어 들인 명령어)
3. 메모리 주소 레지스터
-메모리 주소
4. 메모리 버퍼 레지스터
-메모리와 주고 받을 값 (데이터와 명령어) 메모리 주소 
-레지스터에 있는 주소값으로 제어신호를 통해 해당 번지로 가서 메모리의 실제 값을 저장하게됨
-이놈이 값을 읽으면 프로그램 카운터가 1을 증가
5. 플래그 레지스터
-cpu가 계산한 부가정보 (연산결과)
6. 범용 레지스터
-다양하고 일반적인 상황에서 자유롭게 사용
7. 스택 포인터 
8. 베이스 레지스터

* 7 , 8은 주소지정 방식임
